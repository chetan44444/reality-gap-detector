# Reality Gap Detector (AI)

An AI-powered web app that analyzes whether a statement is realistic or unrealistic.

## Features
- ML-based prediction
- Rule-based fallback
- Confidence score
- Reality score with explanation

## Tech Stack
- Python
- Scikit-learn
- Streamlit
